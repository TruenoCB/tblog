# Halo 2 Plugins

This directory is intended for custom Halo 2 plugin development.

## Creating a new plugin

1. You can clone an existing Halo 2 plugin template or start from scratch using the official Halo 2 Plugin Starter.
2. Develop your custom components, routes, and backend business logic here.
3. Build the plugin and upload the resulting `.jar` file to the Halo 2 console.

For more details on plugin development, refer to the [Halo 2 Plugin Development Guide](https://docs.halo.run/developer-guide/plugin/introduction).
